## v0.1.0 - Jan 23, 2026

![Homepage](https://ihi2erbnrb7tmrfy.public.blob.vercel-storage.com/changelog/homepage.jpg)

# Homepage

A redesigned homepage with clearer pathways for both funders and organizations.
**Improved navigation** and a new FAQ section help users understand what Bumicerts are and how to get started.

![Email Sign-Up](https://ihi2erbnrb7tmrfy.public.blob.vercel-storage.com/changelog/email-signup.jpg)

# Email Sign-Up

Users can now register and log in using just an email and invite code - no crypto wallet required.
A **simpler onboarding experience** for nature stewards and communities everywhere.

![Organization Profile](https://ihi2erbnrb7tmrfy.public.blob.vercel-storage.com/changelog/org-profile.jpg)

# Organization Profile

Every Bumicert is tied to an organization. Users can now build out their organization profile, sharing their story with funders.
**More transparency** about who's doing the work and where it's happening.

![Bumicert Application](https://ihi2erbnrb7tmrfy.public.blob.vercel-storage.com/changelog/bumicert-app.jpg)

# Bumicert Application

Creating a Bumicert is easier than ever. New features include:

- real-time preview
- character limits
- helpful tips for each section
- save-as-draft functionality
- upload GeoJSON files or draw site boundaries directly
  ... and more!
